dobot-arkit-teleop
==================

Teleoperate a Dobot Magician using the iPhone. This uses the Augmented Reality (ARKit) tools in iOS to track the 3D position of the iOS device in the room and moves the Dobot arm in lockstep.

Requirements
------------

- Python 3.6
- iOS >= 13
- Xcode 11.4
- pydobot 1.1.0
